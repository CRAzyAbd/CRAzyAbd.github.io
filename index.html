NIGHTWATCH

    
An ML-powered web application firewall. Built like a product, not a coursework demo.

    

      

        
Modular by design. A regex detection engine covers the five attack classes that matter; a custom rule system extends it without touching the core; real-time traffic analysis keeps it honest; and a machine-learning layer backs up what pattern matching alone would miss.

        

          

            01
            
Detection engine
Regex signatures across SQL injection, XSS, CSRF, LFI and RCE — written, tested, tuned.

          

          

            02
            
Custom rule system
Write, load and version your own signatures. The engine stays untouched.

          

          

            03
            
Real-time analysis
Inspects traffic as it moves — verdicts at request time, not in tomorrow's report.

          

          

            04
            
ML classification
Machine learning sits behind the rules to flag what static patterns can't.
